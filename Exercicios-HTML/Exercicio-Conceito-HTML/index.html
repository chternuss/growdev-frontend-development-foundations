Introdução HTML

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para estruturar e exibir conteúdo na web. Ele define os elementos que compõem uma página da internet, como títulos, parágrafos, links, imagens, tabelas e formulários.

Principais TAGS de Formatação

<b> Representa um texto em negrito, mas sem adicionar significado semântico ao conteúdo. Isso significa que a formatação é puramente visual e não indica importância ou destaque para leitores de tela ou mecanismos de busca.

<strong> É usada para dar ênfase semântica ao texto, tornando-o negrito e indicando que o conteúdo é importante. Além da formatação visual, ela também é interpretada por leitores de tela e mecanismos de busca, ajudando na acessibilidade e no SEO.

<i> É usada para exibir texto em itálico, mas sem adicionar significado semântico. Ela é utilizada principalmente para estilizar palavras, como termos estrangeiros, nomes científicos ou partes do texto que precisam se destacar visualmente.

<em> É usada para aplicar itálico ao texto, mas com ênfase semântica. Isso significa que, além da formatação visual, a palavra ou frase dentro dessa tag é interpretada como algo que deve ter destaque, ajudando na acessibilidade e no SEO.

<mark> É usada para destacar um trecho de texto, geralmente aplicando um fundo colorido (amarelo por padrão nos navegadores). Essa marcação pode ser útil para enfatizar partes importantes de um texto, como resultados de pesquisa ou informações relevantes.

<small> É usada para exibir um texto em um tamanho menor do que o padrão do restante do conteúdo. Normalmente, ela é utilizada para informações secundárias, como notas de rodapé, avisos legais ou detalhes menos importantes.

<del> É usada para indicar que um texto foi excluído ou descontinuado. O conteúdo dentro dessa tag é geralmente exibido com um risco no meio (strikethrough), ajudando a mostrar modificações, correções ou remoções em um texto.

<sub> É usada para exibir texto em subscrito, ou seja, um texto menor e posicionado abaixo da linha base do restante do conteúdo - H2O. Ela é comumente utilizada para fórmulas químicas, expressões matemáticas e notas de rodapé.

<sup> É usada para exibir texto em sobrescrito, ou seja, um texto menor e posicionado acima da linha base do restante do conteúdo - Ex: A fórmula para área de um quadrado é lado2. Ela é comumente utilizada para expressões matemáticas, notação de potência e outros casos onde é necessário elevar um caractere ou número.
