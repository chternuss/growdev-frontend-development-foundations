Exercício 1

Utilizando CSS inline aplique as seguintes regras CSS:
1. Adicione uma cor de fundo com o valor do tipo
Hexadecimal no corpo da página;
2. O título principal da página (h1) deve estar em caixa alta
(maiúsculo). Utilize CSS inline para aplicar a
transformação;
3. Nos títulos de cada seção (Formação, Experiências
Profissionais e Contato), aplique:
a. Um tamanho de fonte de 24px;
b. Uma cor de fundo com o valor do tipo RGB;
4. No parágrafo do rodapé da página, aplique:
a. Um tamanho de fonte de 14px;
b. Uma cor de texto com o valor do tipo Nome para
qualquer tom de cinza;
c. Alinhe o texto ao centro da página;

Exercício 2

Utilizando CSS interno aplique as seguintes regras CSS:
1. Modifique a família de fonte para TODOS os elementos da
página utilizando o seletor CSS correto;
2. Adicione o atributo class com o valor "titulo-secao" nos
elementos que contemplam os títulos de cada seção
(Formação, Experiências Profissionais e Contato). Aplique os
seguintes estilos para os elementos que possuírem a classe
"titulo-secao":
a. Um tamanho de fonte de 24px;
b. Uma cor de fundo com o valor do tipo RGB;
3. Adicione o atributo id com o valor "texto-copyright" no
parágrafo do rodapé da página. Aplique os seguintes estilos
para o elemento que possuir o id "texto-copyright":
a. Um tamanho de fonte de 14px;
b. Uma cor de texto com o valor do tipo Nome para
qualquer tom de cinza;
c. Alinhe o texto ao centro da página

Exercício 3

Utilizando CSS externo aplique as seguintes regras CSS:
1. Aplique todas as estilizações solicitadas no exercício 2, agora
utilizando CSS externo;
2. Adicione o atributo class em cada section da página com os
valores: “formacao”, “experiencias-profissionais” e “contato”
3. Apenas aos parágrafos que são filhos diretos do elemento
section que com a classe "formacao", aplique a transformação
de texto para caixa alta (maiúsculas);
4. Adicione o atributo class com o valor "descricao-cargo" apenas
nos elementos de parágrafo da seção
"experiencias-profissionais" que contém a descrição do cargo;
5. Modifique a cor do texto dos elementos que possuírem a classe
"descricao-cargo" e que sejam filhos diretos de um elemento
com a classe "experiencias-profissionais";
6. Remova o sublinhado dos textos dos links (<a>) contidos na
seção de contato. Assegure que os estilos só serão aplicados
para os links que estiverem dentro do elemento de classe
"contato";
